# Simple Mailer (Node.js + Express)


## Qué hay aquí
API mínima en Node.js para enviar correos mediante SMTP usando nodemailer. Contenedor Docker y fichero docker-compose incluidos.


## Variables de entorno
Copia `.env.example` a `.env` y rellena con los datos de tu servidor SMTP.


## Construir y ejecutar
```bash
docker compose up --build -d
